## Deskripsi Produk

Source code ini merupakan implementasi **Convolutional Neural Network (CNN)** untuk klasifikasi citra daun tomat berbasis **Python dan TensorFlow/Keras**.  
Kode dirancang dengan struktur yang jelas dan mudah dipahami, sehingga dapat digunakan untuk keperluan **pengembangan, penelitian, maupun komersialisasi**.

Model dilatih menggunakan dataset citra yang dibagi menjadi **training** dan **validation** untuk memastikan proses pembelajaran berjalan optimal dan terkontrol.

---

## Fitur Utama
- Implementasi CNN untuk klasifikasi citra multi-kelas
- Struktur kode rapi dan mudah dikembangkan
- Proses training dan validation terpisah
- Visualisasi akurasi dan loss selama training
- Penyimpanan model dalam format `.h5`
- Mudah diintegrasikan ke sistem lain
